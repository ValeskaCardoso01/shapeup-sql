# Sistema de Gestão para Academia Fitness — ShapeUp (Mini-mundo SQL)

Este repositório contém os scripts SQL utilizados para criação e manipulação de dados no banco de dados do projeto ShapeUp.

## Scripts incluídos
| Script | Descrição |
|--------|----------|
| 1_create_tables.sql | Criação das tabelas e chaves |
| 2_insert_data.sql | Inserção inicial dos dados |
| 3_select_queries.sql | Consultas usando SELECT, JOIN, WHERE, ORDER BY |
| 4_updates.sql | Atualizações de dados |
| 5_deletes.sql | Exclusões condicionais |

## Objetivo
Demonstrar domínio da DML (INSERT, SELECT, UPDATE e DELETE) em banco de dados relacional baseado em um projeto real.

## Ferramenta utilizada
MySQL Workbench ou PGAdmin

## Execução
Execute os scripts na ordem numérica indicada.
